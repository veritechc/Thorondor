# Contributing to Thorondor

Thank you for your interest in Project Thorondor.

## Mission

Help create an open-source, modular, mostly 3D-printable wind-powered
platform.

## We welcome contributions in:

-   CAD (Fusion 360, STEP)
-   3D printable parts
-   Mechanical engineering
-   Testing and validation
-   Documentation
-   Electronics
-   Firmware
-   Ideas for new modules

## Design Guidelines

-   Fit within a 256×256×256 mm build volume whenever practical.
-   Prefer standard metric hardware.
-   Use commercially available bearings and fasteners.
-   Design for repairability.
-   Document all design decisions.

## Pull Requests

Please include: - Description of the change - Photos or renders if
applicable - Test results - Updated documentation
