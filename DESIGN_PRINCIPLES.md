# Thorondor Design Principles

## Core Philosophy

Thorondor is a platform---not a single machine.

### Principles

1.  Modular first.
2.  Mostly 3D printable.
3.  Consumer printer friendly.
4.  Standardized hardware.
5.  Easy to repair.
6.  Beautiful engineering.
7.  Open collaboration.

## Standards

-   Print envelope: 256×256×256 mm
-   Metric fasteners preferred
-   Stainless hardware outdoors
-   Weather resistant materials (ASA/PETG preferred)

## Long-Term Goal

Create a universal mechanical interface allowing interchangeable
wind-powered modules.
