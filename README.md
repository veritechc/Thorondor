# Thorondor

> **Harnessing Wind. Empowering Innovation.**

**Project Thorondor** is an open-source engineering initiative by
**Sillmiron Studios LLC** to create a modular, mostly 3D-printable
wind-powered mechanical platform.

Rather than designing a single-purpose windmill, Thorondor is being
developed as a reusable platform capable of driving interchangeable
mechanical modules. The first application is a pond aeration system, but
the long-term vision includes water pumping, electrical generation,
environmental monitoring, and other wind-powered tools.

------------------------------------------------------------------------

## Vision

Build a beautiful, repairable, expandable, and affordable wind-powered
system that anyone can manufacture using consumer 3D printers and
commonly available hardware.

### Core Principles

-   Modular by design
-   Mostly 3D printable
-   Open-source hardware
-   Easy to repair
-   Easy to modify
-   Built using readily available components
-   Scalable from desktop prototypes to full-size installations

------------------------------------------------------------------------

# Current Objective

Develop a reliable wind-powered pond aeration system using:

-   Helical Savonius rotor
-   Modular drive head
-   Diaphragm air pump
-   Bottom diffuser aeration

Future versions will support additional interchangeable modules without
redesigning the tower or rotor.

------------------------------------------------------------------------

# Design Standards

Every major component should:

-   Fit within a 256 × 256 × 256 mm print volume
-   Be printable on consumer-grade 3D printers
-   Use standardized bearings and hardware
-   Be field-serviceable with basic tools
-   Minimize proprietary components

------------------------------------------------------------------------

# Open Source Philosophy

Thorondor is intended to be an open engineering platform that encourages
experimentation, collaboration, and community contributions.

Whether your goal is to aerate a pond, pump water, generate electricity,
or invent something entirely new, we hope Thorondor provides a
foundation you can build upon.
